This repository does not currently include repo-local browser automation for screenshot capture. The following screenshots should be captured manually from the live app before submission.

1. Input / `Example hypotheses` page
2. Literature QC panel
3. Plan overview
4. Materials and budget view
5. Sources tab
6. Review tab
7. Readiness / system status panel
